# Editorial Dawn Wedding — Website Template

![Screenshot](../images/lovable/editorial-dawn-wedding.jpg)

## Overview
A magazine‑style wedding one‑pager with cinematic hero imagery, refined serif typography, and editorial sectioning. It blends story, countdown, event details, and RSVP into a premium, fashion‑editorial layout.

## Layout
- **Hero**: full‑bleed couple image + large serif names, date line, and location.
- **Story**: timeline of five moments with image + text pairings and a pull quote.
- **Countdown**: “The Big Day” timer block with days/hours/minutes/seconds.
- **Event Details**: three cards (Ceremony, Cocktail Hour, Reception) with venue copy.
- **RSVP**: compact form (name, email, attendance radio, note) + CTA.
- **Footer**: date/location line + closing note.

## UX patterns
- Anchor nav (Our Story / Details / RSVP) for quick jumps.
- Alternating image/story blocks for rhythm.
- Countdown adds urgency/anticipation.

## Animations
Likely smooth scroll and subtle fades; relies on imagery and typography for drama.

## Visual style
- Dark, cinematic hero image with cream typography.
- Editorial serif headings with generous spacing.
- Soft neutral background + large image blocks.

## Components
- Top nav + anchors
- Hero image + title block
- Timeline cards
- Countdown widget
- Event cards
- RSVP form + CTA
